**AIM:** Identify the spam tweets among corpus of tweets using similarity among textual data.

PROGRAMMING LANGUAGE USED: Python 3.x

IDE: Jupyter Notebook

**DISTANCE MEASURES:**

1. Jaccard Distance

2. Dice Coefficient

3. Cosine Similarity

4. Euclidean Distance

5. Levenshtein Distance
